# 🎓 EduAgent - AI Study Companion

An AI-powered study companion built for the Microsoft Agents League Hackathon.

## 🚀 Features
- **Explain Topic** — Get simple explanations of any topic
- **Generate Quiz** — Auto-generate 5 MCQ questions
- **Study Plan** — Get a 7-day personalized study plan

## 🛠️ Built With
- HTML, CSS, JavaScript
- Claude AI API (Anthropic)
- GitHub Copilot for AI-assisted development

## 💡 How to Use
1. Enter any topic in the search box
2. Choose Explain, Quiz, or Study Plan
3. Click Go and get instant AI response!

## 🎯 Use Case
Helps students learn any topic quickly using AI — explanations, quizzes, and study plans in seconds!
